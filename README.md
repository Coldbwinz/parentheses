Пример поиска валидных скобок

Задание:
**Найти валидные круглые скобки. 
Обязательно счетчик и правильные скобки нужно выводить.** 

Примеры:
1. Ввод: "(()" Вывод: 2 - () 
2. Ввод: ")()())" Вывод: 4 - ()() 
3. Ввод: ")(()())" Вывод: 6 - (()()) 
4. Ввод: ")(" Вывод: 0 
5. Ввод: "())(()())(()" Вывод: 10 - ()(()())()
